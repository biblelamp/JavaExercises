Сборка Java проектов: быстрый взгляд на Ant, Maven и Gradle

Умение работать с системами сборки необходимо каждому Java developer. Цель вебинара — получить представление о таких системах как Apache Ant, Apache Maven и Gradle. В ходе вебинара мы опробуем все три на простом Java проекте, проведя компиляцию, запуск, тестирование и сборку jar файла. Нам понадобятся установленные и настроенные:

- Java SE DK 8 http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
- Apache Ant http://ant.apache.org/
- Apache Maven https://maven.apache.org/
- Gradle https://gradle.org/

Желательны также базовое знание Java и умение работать в командной строке. Материалы вебинара (исходные коды и сборочные сценарии) будут доступны по ссылке https://github.com/biblelamp/JavaExercises/AntMavenGradle
